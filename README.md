# E-Commerce Web Application

Personal practice project using React, React-Redux, and Firebase.

**Live Demo:** [Famous Mooncake E-Commerce](https://famous-mooncake-513849.netlify.app/)

## Features
- User Authentication with Firebase
- Product Catalog
- Shopping Cart
- Checkout System
- Responsive Design

## Tech Stack
- React
- React-Redux
- Firebase
- Netlify
